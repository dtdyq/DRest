1.Intro

DRest just for http apis.on main frame,each tab meains a rest collection,called rest entity in drest.entity contains a list of rest item,each item means a http api.

2.Func

  1. file menu:open existed rest entity;save current rest entity;save all rest entities;edit global variable(can be used in url/body)

  2. help menu:show download tasks

  3. rest item supported operations:

    • rename the item

    • add new rest item

    • insert new rest item on bottom of selected items

    • duplicate selected rest items

    • delete selected rest items

    • run selected rest items :parallel and sequential

    • config basic auth for all selected items or for all in current rest entity

    • config proxy for all selected items or for all in current rest entity

    • config ssl verify

  1. mouse right click:

    • on method column:batch set rest item method
    • on url column:batch edit rest item url
    • on head column:batch edit for header or form
    • on status column:classify rest items by response status
    • on cost column:sort by request cost time and set const time unit(ms default)

  2. up load file?

    set 'upload' as form key and file path as form value to enable file upload

  3. copy across different rest entity?

    try to use ctrl+c and ctrl+v

  4. double click on rest item to see the detail function of it,like download .etc

  5. use ctrl+z to undo the latest operation or open operation log pane to rollback to history operatons
